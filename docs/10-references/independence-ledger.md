# Independence Ledger

This ledger summarizes the public independence rules without publishing private audit details.

## Public Rules

- Legacy examples were removed from the public repository.
- Adapted concepts were re-derived from the Local SEO Knowledge Base, 2025-2026 PDFs, or verified web sources.
- Unsupported numerical claims were dropped.
- Contradicted claims were replaced with verified values and citations.
- Public prompts use a standardized schema and freshly written instructions.
- Public examples are generic or newly created for this repository.

## Acceptance Standard

A public file passes only when it can be understood without private notes, private examples, or unverifiable statistics. The detailed internal scan remains outside the public repository.
