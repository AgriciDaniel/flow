## [1.0.0] — 2026-04-25

Initial public release.

### Added
- **FLOW framework** (Find → Leverage → Optimize → Win) — see `docs/01-framework/flow-framework.md`
- **AI Search Surface Map** — how Google, AI Overviews, ChatGPT, Perplexity, Reddit, YouTube, LinkedIn, and local surfaces interact in 2026
- **Foundations pillar** — search intent, E-E-A-T, helpful content, technical and schema basics
- **Find pillar** — keyword research, audience avatar, query fan-out and topical relevance, geographical relevance
- **Leverage pillar** — distributed presence, local citations and links, Reddit + LinkedIn + community
- **Optimize pillar** — GEO formatting and extraction, entity consistency and LLM tracking, content and CTR optimization
- **Win pillar** — BOFU and conversion content, PPC and first-party intelligence, dual-surface scorecard
- **Local SEO pillar** (separate from F/L/O/W) — overview, Google Business Profile guide, local content and service pages, local audits
- **Playbooks** — SaaS, Ecommerce, Service Business, B2B Lead Generation, Affiliate-Content
- **Prompt library** — 42 standardized AI prompts across Find / Leverage / Optimize / Win / Local stages, including 3 net-new Win prompts (BOFU Page Brief Generator, Conversion Audit, Dual-Surface Content Scorecard)
- **8 FLOW-native diagrams** + GitHub social preview image
- **Obsidian vault mirror** at `obsidian-vault/` with wikilinks and frontmatter
- **`llms.txt` + `llms-full.txt`** for AI agent ingestion (llmstxt.org spec)
- **Repo standards** — README, LICENSE (CC BY 4.0 + MIT), CONTRIBUTING, CODE_OF_CONDUCT (Contributor Covenant 2.1), SECURITY, CHANGELOG
- **GitHub workflows** — markdown-lint, link-check, docs evidence-standard check
- **Issue templates** — bug report, source correction, feature request

### Evidence layer
- 15 verified statistics with full URL + retrieval date provenance in `docs/10-references/stats-provenance.json`
- 4 contradicted statistics replaced with verified alternatives
- 8 unverifiable statistics dropped per the evidence policy
- Bibliography of 2026 sources from Search Engine Journal, Hobo Web, CallRail, Pew Research, Ahrefs, Semrush, Backlinko, BrightLocal, seoClarity, StatCounter, and others

### Independence
- Re-derived from independent 2025–2026 sources following the v4 independence rule
- Zero references to legacy case studies from the source-influence book
- Acknowledgment of *Ski Slope Strategy 3.0* by Chris Von Wilpert (Content Mavericks) as influence in `docs/10-references/attribution.md`
