# Prompt Library

Every prompt in this folder uses the same schema:

- Use This When
- AI Compatibility
- Inputs
- Prompt
- Output
- Example
- See Also

The library is staged by FLOW: Find, Leverage, Optimize, Win, and Local. Prompt logic is rewritten for this public repository from independent source material and field-use patterns. Do not treat the prompts as magic strings; treat them as structured thinking tools.
